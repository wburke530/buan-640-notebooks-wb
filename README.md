# BUAN 640 Course Notebooks and Data Files

This repository contains **course notebooks and data files** used in BUAN 640 (Applied Statistics).

The primary purpose of this repository is to share instructional materials with students.  
Notebooks can be viewed and run in Google Colab or a local Jupyter Notebook environment.

⚠️ **Work in progress**  
Materials in this repository may be updated throughout the semester.  
File names, structure, and content may change as the course evolves.

---

## How to use these materials

- Open any notebook using Google Colab
- You can run cells and experiment freely
- If you would like to make changes, save a personal copy to your own Google Drive or local machine

Students are not required to submit anything via this repository.

---

## Repository structure

- `notebooks/` – Jupyter notebooks used in class
- `data/` – Example datasets used by the notebooks
